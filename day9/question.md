Почему проверка расширения файла полезна, но недостаточна для безопасной обработки загружаемых файлов?  
- файл может быть замаскированным или содержать какой-либо payload  
- из примеров недавних https://www.securitylab.ru/news/571386.php (уязвимость работает сразу при открытии файла, без каких либо действий пользователя)
- двойные расширения (косяки валидатора (если он не умеет такие случаи обрабатывать))
- с теми же xml с инхекцией XXE (вроде как работать с файлом нужно, но без дополнительных мер защиты можно помать XXE)
