```
from flask import Flask, request, redirect

app = Flask(__name__)

@app.route("/go")
def go():
    url = request.args.get("next")
    return redirect(url)

if __name__ == "__main__":
    app.run(debug=True)
```

# Вопросы:

### Где здесь может быть уязвимость?
- Open Redirect (строки 8-9)
### Как её можно эксплуатировать?
- из параметра next приходит ссылка как plaintext без какой-либо обработки. по указанной ссылке и будет совершен редирект
### Какие последствия?
- перенаправление пользователя на поддельный сайт внешний урл без проверки  
https://ozon.ru/redirect?url=https://0z0n.ru  
- CSRF  
- репутационные издержки для ресурса  
### Как исправить?
- отказаться от пользовательского ввода  
- использовать белые спсики разрешенных к редиректу доменов  
- если редирект в рамках одного домена, то можно включить проверку same origin
